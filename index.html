Project Description

The purpose of this project is to gather bacterial data from a sampling of volunteers' belly buttons for analysis on the development of Improbable Beef's next synthetic beef. There is potential for a bacteria found growing in a human belly button to produce a protein that could be used to grow a synthetic beef substitute.

This dashboard allows a volunteer to select their ID number and then view a series of graphs displaying the bacteria found in their belly button sample.
